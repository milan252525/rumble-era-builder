What is Rumble Royale?

Rumble Royale is the #1 Battle Royale simulator on Discord! This page serves as a tool for creating custom eras. For more information about Rumble Royale, please visit our website or join our Discord server!

Custom Eras

Rumble Royale battles can be started in various eras, each era features unique weapons, actions, and events. In the Futuristic era, you might find yourself in a shootout with a blaster while aliens invade the arena! In the Magical era, you could encounter dragons swarming the arena while you search for your lost wand!

Official eras, created by our team, are unlockable for everyone, with season passes and progression tied to them. Custom eras allow you to personalize and create unique experience for your server. They can reflect your server's unique interests, fun memories, or passions. A game your members play? A music band they love?

Custom Era Creation Walkthrough

Ever played Rumble and thought, "I wish this had a Cake era"? No? Let's create it together anyway!

Main Information

Start by entering a short and clear era name. Then, pick a color for the Discord embeds using the color circle on the right. We have to go for pink here... Next, select a nice cake image as an icon and add a corresponding emoji. Avoid using Discord links for images, they might expire. Instead, use a hosting service like Imgur. For emojis, use a standard one from your keyboard. You can also use a custom Discord emoji. But you need to get its ID. To get it, go to Discord and send \ (backslash) followed by the emoji. It should look like <:cake:889150097873576006>. If you choose to use a Discord emoji, please make sure the bot is in the same server, otherwise it will not display properly.

Phrases

Now it's time to unleash your inner writer! By now you must have noticed the bot fills in player names, titles and pronouns. You have to specify where they go. But don't worry, it's quite straightforward. Different types of phrases require different patterns filled in. Kill phrases need two players, while revive phrase only require one. Each section contains a description of what to include. Toggle the "Show pronouns formatting" button for a handy formatting guide that's always on your screen. If you miss something, the website will display a warning and let you know what to fix.

Items

Every battle needs some fun and deadly weapons. Let's add a cake bomb! Enter the name and hit "Add Item". Choose some cool emoji, like we have already done once. You will need to include two types of phrases. Item Unlock Phrases for when a player gets the item, and Item Kill Phrases for when it's used to make a kill.

Remember to follow the formatting. For example, Item Unlock phrases requires you to use "__{item}__" instead of the item name.

Mass Events

There are two types of mass events you can add to your era. One kills and the other revives a higher number of players. Each requires a name and an image icon. Mass events are optional, and you can leave them empty. However, if you create one type of mass event, you must create one for the other type as well. Otherwise, they won't appear in battle.

Discord Setup

It's time to upload your new era to the bot. Click the big Download Era button to download a file ending in .json. Never edit this file manually. If you ever want to make changes, upload the file back to the website using the Upload Era File button. With your file ready, go to Discord and use /custom upload slash command. Attach the file.

If you are updating it, fill in the replace argument.

Your era is now ready to be started! Unlike regular eras, custom eras must be started with the /battle-custom command. It works like the regular one, allowing you to start a normal or staff battle.
